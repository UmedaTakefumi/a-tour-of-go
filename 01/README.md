Go01：Hello, 世界
------
### 問題 ###
コンソールにHello, 世界と表示する

### ソースコード ###

    package main
    
    import "fmt"
    
    func main() {
      fmt.Println("Hello, 世界")
    }

### 実行結果 ###

    $ ./hello.go 
    Hello, 世界
    $
